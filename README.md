# Project Eldrin

Public-facing repository for the Project Eldrin story world.

This repo is for released material only. It contains curated chapters, spoiler-safe lore, public character profiles, bestiary entries, release notes, and changelog updates for the community.

## What You Can Expect Here

- released story chapters
- public worldbuilding notes
- partial character profiles
- bestiary entries
- release notes

## Canon Policy

Community theories and suggestions are welcome, but the author remains the only source of canon.
